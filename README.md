# Kabug.rb
Repositório do projeto Kabug com Cucumber, Capybara e Ruby

## Step by Step

### Instalar o Ruby
`
sudo apt install ruby-full
`

### Instalar o Bundler
`
gem install bundler
`

### Instalar as dependências do projeto ruby
`
bundle install
`

### Executar localmente
`
bundle exec cucumber
`

### Executar no servidor de CI
`
bundle exec cucumber -p ci
`
